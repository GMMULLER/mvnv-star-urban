About

This is a companion website for the STAR Report on Multivariate Network Visualization Techniques. The STAR report can be found here.

Multivariate networks capture relationships between nodes and attributes about the nodes and relationships themselves. Most relational data is associated with rich attributes. Visualization of multivariate networks, however, is challenging, especially when both the topology of the network and the attributes need to be considered concurrently. In this state-of-the-art report, we analyze current practices and classify techniques along four axes: Layouts, View Operations, Layout Operations, and Data Operations. We also provide an analysis of tasks specific to multivariate networks and give recommendations for which technique to use in which scenario. Finally, we briefly survey application areas and evaluation methodologies. Keywords: graph visualization, network visualization, multivariate networks.
